# NAME
vchmod - visual chmod tool

# SYNOPSIS
**vchmod** *file* [file...] [chmod options]

# DESCRIPTION
Same usage as chmod, but you don't need to specify the permission value. The program provides a visual interface to select permissions interactively.

If multiple files are provided, the initial permission display is based on the first file.
